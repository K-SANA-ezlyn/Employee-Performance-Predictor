# Employee Performance Predictor using Data Analytics

## Project Overview

This project predicts employee performance using Machine Learning and HR Analytics.

The goal is to classify employees into:

* High Performer
* Medium Performer
* Low Performer

based on employee attributes like attendance, training hours, salary, manager feedback, work-life balance, projects completed, and experience.

This helps HR teams and managers make data-driven decisions for promotions, training, employee retention, and performance improvement.

---

## Problem Statement

Traditional employee performance evaluation often depends heavily on annual reviews and manager judgment, which may introduce bias and delays.

This project builds a predictive system that helps organizations identify performance patterns early using data analytics and machine learning.

The system helps answer:

* Which employees are likely to perform well?
* Which employees may need support or training?
* How can HR improve employee development strategies?

## Dataset Features

The dataset contains the following features:

| Feature            | Description                           |
| ------------------ | ------------------------------------- |
| Age                | Employee age                          |
| Experience         | Total years of experience             |
| Department         | Employee department                   |
| Salary             | Monthly/Annual salary                 |
| Training_Hours     | Total training hours completed        |
| Attendance         | Attendance percentage                 |
| Projects_Completed | Number of completed projects          |
| Overtime_Hours     | Extra working hours                   |
| Manager_Feedback   | Manager rating score                  |
| Work_Life_Balance  | Employee work-life balance score      |
| Performance_Rating | Target variable (High / Medium / Low) |
